This work has practical implications for efficient model composition and transfer learning in the foundation model era, enabling lightweight adapter-based integration without full model retraining.

One-Step Flow Policy: Self-Distillation for Fast Visuomotor Policies

cs.AI updates on arXiv.org · 2026-03-16

💼 25 ⚡ 25

One-Step Flow Policy (OFP) introduces a self-distillation framework that compresses iterative flow/diffusion models into single-step action generation for robotic control, eliminating the inference latency that degrades real-time manipulation performance. The method combines self-consistency loss (enforcing coherent temporal transport), self-guided regularization (mode-sharpening), and warm-start mechanisms to achieve competitive or superior performance versus 100-step diffusion policies across 56 manipulation tasks. This addresses a critical bottleneck in generative visuomotor policies: practitioners can now leverage the expressiveness of continuous, multimodal distributions without sacrificing control frequency.

Cost-Efficient Multimodal LLM Inference via Cross-Tier GPU Heterogeneity

cs.AI updates on arXiv.org · 2026-03-16

💼 28 ⚡ 78

This paper identifies an optimal partitioning strategy for MLLM inference across heterogeneous GPUs by exploiting the modality boundary between vision encoding (compute-bound) and language generation (memory-bandwidth-bound), reducing cross-device transfer from GB-scale KV caches (O(L * s_ctx)) to MB-scale embeddings (O(N_v * d))—an O(L) complexity improvement that scales with model depth. The key insight enables disaggregation on lower-bandwidth interconnects than existing stage-level approaches (avoiding NVLink dependency), making multi-GPU MLLM serving significantly more cost-efficient for heterogeneous hardware setups. This has immediate practical implications for deploying MLLMs on commodity hardware clusters where compute-optimized and memory-optimized GPUs can be paired without expensive high-bandwidth networking.
