Sparrow: Text-Anchored Window Attention with Visual-Semantic Glimpsing for Speculative Decoding in Video LLMs

cs.AI updates on arXiv.org · 2026-02-18

💼 18 ⚡ 78

Sparrow addresses the critical failure of speculative decoding in Video LLMs by leveraging a key insight: visual semantics are implicitly encoded into text hidden states during inference, making raw visual inputs redundant in deeper layers. The framework employs text-anchored window attention with hidden state reuse to offload visual computation to the target model while using intermediate-layer visual state bridging to train the draft model with semantically-rich representations, effectively circumventing attention dilution and KV cache explosion. This approach enables practical acceleration of Video LLM inference while maintaining performance—crucial for real-world deployment of long-context video understanding systems.

MARS-Sep: Multimodal-Aligned Reinforced Sound Separation

cs.AI updates on arXiv.org · 2026-02-18

💼 23 ⚡ 87

MARS-Sep reformulates universal sound separation as a reinforcement learning decision-making problem using a factorized Beta mask policy guided by a multimodal preference reward model, directly addressing the core misalignment where traditional signal-metric optimization produces semantically contaminated outputs despite low-level accuracy. By leveraging an audio-text-vision encoder for reward signals, the approach achieves better perceptual quality and semantic consistency across text-, audio-, and image-queried separation tasks compared to conventional mask regression methods. This preference-alignment perspective mirrors recent LLM techniques, offering a principled way to optimize for human-perceived quality rather than just reconstruction fidelity in complex acoustic scenarios.

💼 22 ⚡ 78

**EventMemAgent** introduces a hierarchical event-centric memory architecture to overcome MLLMs' context window limitations in streaming video understanding, using dual-layer short-term (event boundary detection with reservoir sampling) and long-term (structured event archival) memory to enable active reasoning over infinite visual streams. This approach addresses the critical trade-off between maintaining long-range context and preserving fine-grained details by organizing memory at event granularity rather than frame-level, enabling more efficient context management for complex downstream tasks. The framework's adaptive tool use capability suggests dynamic selection of processing strategies based on streaming content, improving computational efficiency for real-time video understanding applications.

Train Short, Inference Long: Training-free Horizon Extension for Autoregressive Video Generation

cs.CV updates on arXiv.org · 2026-02-18

💼 23 ⚡ 78

This paper addresses temporal degradation in autoregressive video diffusion models when generating beyond training horizons by identifying spectral bias in 3D positional embeddings as the root cause. FLEX proposes a training-free solution combining Frequency-aware RoPE Modulation (adaptively interpolating low-frequencies while extrapolating high-frequencies) with Antiphase Noise Sampling to inject dynamic priors, enabling length extension without retraining. The approach is practically significant for enabling efficient long-form video generation from models trained on shorter sequences, with potential applications across content creation and scientific visualization.

Panini: Continual Learning in Token Space via Structured Memory

cs.AI updates on arXiv.org · 2026-02-18

💼 28 ⚡ 32

Panini introduces a continual learning framework that replaces traditional RAG's inefficient document retrieval with Generative Semantic Workspaces (GSW)—structured semantic memory that consolidates new documents into an entity and event-aware network while keeping the base model frozen. This approach reduces redundant reasoning over repeated documents and mitigates irrelevant context injection, improving both computational efficiency and generation fidelity during inference. The method demonstrates a human-like learning paradigm where knowledge accumulation happens externally through semantic consolidation rather than through parametric model updates.
