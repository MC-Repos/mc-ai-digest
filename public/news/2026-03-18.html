DASH: Dynamic Audio-Driven Semantic Chunking for Efficient Omnimodal Token Compression

cs.AI updates on arXiv.org · 2026-03-18

💼 24 ⚡ 92

DASH introduces a training-free approach to multimodal token compression by using audio embeddings as semantic anchors to detect boundary discontinuities via cosine similarity, enabling dynamic variable-length segmentation that preserves the inherent structure of audio-visual streams rather than relying on fixed partitioning. This cross-modal segmentation strategy allows selective token retention within semantically coherent segments, significantly reducing inference costs for OmniLLMs while maintaining contextual integrity that fixed-window and attention-based pruning methods fail to capture.

Conflict-Aware Multimodal Fusion for Ambivalence and Hesitancy Recognition

cs.CV updates on arXiv.org · 2026-03-18

💼 28 ⚡ 87

ConflictAwareAH tackles ambivalence/hesitancy detection by inverting the typical multimodal fusion strategy—treating cross-modal disagreements as primary signals rather than noise, using pairwise conflict features (element-wise differences between video, audio, and text embeddings) as bidirectional indicators where large conflicts flag A/H states while consistency anchors negative samples. This approach directly addresses the false-positive bias of text-dominant methods, making it particularly valuable for clinical applications where subtle affective contradictions (e.g., verbal agreement with contradictory facial expression) carry diagnostic significance.

Diffusion Models for Joint Audio-Video Generation

💼 28 ⚡ 87

This addresses a critical gap in multimodal generation where individual modalities are well-developed but synchronized audio-video synthesis remains largely unexplored, with implications for content creation, game design, and synchronization-dependent applications.

When Thinking Hurts: Mitigating Visual Forgetting in Video Reasoning via Frame Repetition

cs.CV updates on arXiv.org · 2026-03-18

💼 28 ⚡ 78

The paper identifies "visual anchor drifting" in video reasoning MLLMs where extended Chain-of-Thought processing causes models to over-rely on generated text and hallucinate, degrading VQA performance. FrameRepeat addresses this with a lightweight repeat scoring module that automatically identifies frames requiring reinforcement during inference, avoiding expensive retraining while maintaining generalizability across architectures. This approach offers a practical, model-agnostic solution to the fundamental tension between reasoning depth and visual grounding in video understanding tasks.

MoLoRA: Composable Specialization via Per-Token Adapter Routing

cs.AI updates on arXiv.org · 2026-03-18

💼 28 ⚡ 78

MoLoRA introduces per-token adapter routing instead of sequence-level routing, enabling dynamic specialization where individual tokens are routed to domain-specific LoRA adapters based on either vocabulary structure (multimodal) or learned gating (semantic), achieving O(N) computational complexity versus O(K·N) for traditional per-sequence approaches. This approach elegantly handles multimodal generation and mixed-capability requests (e.g., code generation for equations) by composing multiple specialized adapters within a single forward pass, with provably optimal efficiency gains.

# Summary HEAR introduces Vision-Sound-Language-Action (VSLA), a continuous control framework that addresses a critical gap in robotic manipulation by treating real-time environmental audio as continuous state feedback rather than static prompts. The framework tackles the "Blind Execution Interval" problem where acoustic events are missed between discrete audio observations during action chunking, enabling robots to catch fleeting auditory cues (e.g., collision sounds, task completion indicators) essential for closed-loop task verification. This paradigm shift from discrete to streaming audio integration represents a meaningful advance toward multimodal embodied AI that leverages all available sensory modalities for robust manipulation under latency constraints.
