EchoTorrent: Towards Swift, Sustained, and Streaming Multi-Modal Video Generation

cs.CV updates on arXiv.org · 2026-02-17

💼 28 ⚡ 87

EchoTorrent addresses the latency and temporal stability challenges in streaming multi-modal video generation through a four-component approach: multi-teacher domain-expert training for knowledge transfer, Adaptive CFG Calibration to eliminate redundant computations via phased scheduling, and hybrid long-tail forcing for alignment—enabling single-pass inference per diffusion step. The method specifically tackles streaming-specific degradation artifacts (spatial blurring, temporal drift, lip desynchronization) that emerge when balancing real-time performance with multi-modal coherence, resolving a critical efficiency-performance trade-off for practical deployment. This represents a significant optimization path for streaming video generation by combining knowledge distillation, adaptive guidance calibration, and architectural constraints rather than architectural redesign alone.

This work highlights the challenge of scaling multimodal retrieval to video's temporal dimension and demonstrates that LLM-augmented search outperforms direct semantic matching for domain-specific creative workflows.

EgoSound: Benchmarking Sound Understanding in Egocentric Videos

EgoSound introduces the first benchmark for evaluating multimodal LLMs' egocentric sound understanding, addressing a critical gap where current vision-language models neglect audio as a crucial reasoning modality. This work highlights that audio provides essential contextual cues for off-screen events and spatial reasoning in egocentric contexts, establishing systematic evaluation methodology for building more perceptually complete multimodal AI systems.

Situation Graph Prediction: Structured Perspective Inference for User Modeling

This work introduces Situation Graph Prediction (SGP), a novel framework that models user perspective states (goals, emotions, contexts) as structured, ontology-aligned graphs inferred from multimodal observable data, addressing the data scarcity problem through synthetic generation where latent labels are aligned with observable traces by design. The researchers demonstrate that perspective inference significantly outperforms surface-level extraction using GPT-4o with retrieval-augmented in-context learning, revealing that latent-state modeling requires fundamentally different approaches than shallow behavioral extraction. This approach has implications for privacy-preserving user modeling systems where direct perspective annotation is infeasible.

VisPhyWorld: Probing Physical Reasoning via Code-Driven Video Reconstruction

cs.AI updates on arXiv.org · 2026-02-17

💼 15 ⚡ 72

VisPhyWorld introduces an execution-based benchmark that evaluates MLLMs' physical reasoning by requiring models to generate executable simulator code from video observations, rather than relying on recognition tasks (VQA/VoE) that can be answered without explicit physical hypotheses. The framework separates physical reasoning from rendering by making inferred world representations directly inspectable, editable, and falsifiable through runnable code. VisPhyBench, containing 209 scenes from 108 physical templates, systematically evaluates both appearance reconstruction and motion plausibility—providing a more rigorous test of genuine physical understanding versus pattern matching.

IDPruner: Harmonizing Importance and Diversity in Visual Token Pruning for MLLMs

cs.AI updates on arXiv.org · 2026-02-17

💼 22 ⚡ 72

IDPruner addresses the computational bottleneck in MLLMs by proposing a principled framework that balances token importance and semantic diversity in visual token pruning, using the Maximal Marginal Relevance (MMR) algorithm to achieve Pareto-optimal solutions without requiring attention maps. This approach advances beyond ad-hoc combinations of importance and diversity metrics by systematically characterizing their trade-offs, enabling more efficient inference while maintaining representational quality. The attention-map-free design makes it broadly applicable across different MLLM architectures and potentially more computationally efficient than attention-dependent pruning methods.

This addresses a critical gap in long-horizon dexterous manipulation where implicit end-effector prediction fails, offering better interpretability and cross-robot generalization for robotic systems.

BitDance: Scaling Autoregressive Generative Models with Binary Tokens

cs.AI updates on arXiv.org · 2026-02-17

💼 18 ⚡ 72

BitDance introduces a novel approach to autoregressive image generation by replacing traditional codebook indices with binary tokens, enabling each token to represent up to 2^256 states through high-entropy latent representations. The key innovation is a binary diffusion head that uses continuous-space diffusion instead of softmax classification for sampling, combined with a "next-patch diffusion" decoding method that parallelize token prediction while maintaining accuracy. This achieves state-of-the-art FID (1.24) on ImageNet 256×256 with 5.4× parameter reduction (260M vs 1.4B) and 8.7× faster inference compared to existing parallel AR models, suggesting a promising direction for efficient and compact generative models.
