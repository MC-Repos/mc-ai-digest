DCER: Dual-Stage Compression and Energy-Based Reconstruction

cs.AI updates on arXiv.org · 2026-02-07

💼 15 ⚡ 87

DCER introduces a dual-stage architecture for robust multimodal fusion: within-modality compression uses frequency transforms (wavelet/DCT) to denoise while cross-modality bottleneck tokens enforce genuine fusion, and energy-based reconstruction recovers missing modalities via gradient descent with learned energy functions providing uncertainty quantification. The approach achieves SOTA results on standard benchmarks while maintaining robustness to both noisy inputs and missing modalities—the energy function's correlation with prediction error (ρ > 0.72) enables practical confidence estimation without additional overhead.

FlashBlock: Attention Caching for Efficient Long-Context Block Diffusion

cs.AI updates on arXiv.org · 2026-02-07

💼 28 ⚡ 78

FlashBlock exploits cross-step attention redundancy in block diffusion by caching block-external attention outputs that remain stable across diffusion steps, while recalculating only block-internal attention that varies significantly. This approach reduces computational overhead and KV cache access for long-context generation in diffusion models without architectural modifications, addressing a key bottleneck in scaling block diffusion for minute-long videos and extended text generation.

ProAct: Agentic Lookahead in Interactive Environments

cs.AI updates on arXiv.org · 2026-02-07

💼 28 ⚡ 15

ProAct addresses LLM agent failure in long-horizon interactive tasks by distilling environment-based search into compact causal reasoning chains via Grounded LookAhead Distillation (GLAD), eliminating inference-time search overhead while preserving lookahead capability. The framework augments this with Monte-Carlo Critic, a lightweight auxiliary value estimator that calibrates policy-gradient algorithms (PPO/GRPO) using efficient environment rollouts, improving decision accuracy without prohibitive computational costs. This two-stage approach effectively transfers complex planning logic into the agent's weights, enabling more reliable multi-step reasoning in interactive environments where error compounding typically degrades performance.

RaBiT: Residual-Aware Binarization Training for Accurate and Efficient LLMs

cs.AI updates on arXiv.org · 2026-02-07

💼 35 ⚡ 35

RaBiT addresses a critical failure mode in residual binarization for LLMs called "inter-path adaptation," where parallel binary paths redundantly learn overlapping features during QAT, degrading error compensation. Rather than heuristic constraints like path freezing, RaBiT enforces a residual hierarchy by sequentially deriving each binary path from shared full-precision weights, ensuring non-redundant feature specialization and improved model expressivity. This approach enables efficient matmul-free inference with extreme quantization while maintaining better accuracy—particularly relevant for edge deployment of LLMs.
