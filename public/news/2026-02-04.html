CamReasoner: Reinforcing Camera Movement Understanding via Structured Spatial Reasoning

cs.AI updates on arXiv.org · 2026-02-04

💼 25 ⚡ 72

CamReasoner addresses a critical gap in video understanding by reformulating camera movement classification from black-box prediction into structured geometric reasoning, using an Observation-Thinking-Answer paradigm that explicitly models spatio-temporal cues like trajectories and view frustums. The framework leverages 18k supervised reasoning chains and 38k RL-aligned samples to ground motion inferences in physics rather than superficial visual patterns, marking the first application of reinforcement learning for logical alignment in camera motion understanding. This approach has significant implications for video analysis, 3D scene understanding, and cinematic AI applications where accurate geometric motion decomposition is essential.

SupCLAP: Controlling Optimization Trajectory Drift in Audio-Text Contrastive Learning with Support Vector Regularization

cs.AI updates on arXiv.org · 2026-02-04

💼 15 ⚡ 85

SupCLAP addresses a fundamental instability in audio-text contrastive learning by introducing Support Vector Regularization (SVR) to constrain the perpendicular component of negative sample gradients, which simultaneously carries useful information yet causes optimization drift. The method uses an auxiliary support vector with an unsupervised semantic radius to balance this trade-off, improving both training stability and downstream performance in multimodal tasks like cross-modal retrieval and multimodal LLM integration.

FastSLM: Hierarchical Frame Q-Former for Effective Speech Modality Adaptation

cs.AI updates on arXiv.org · 2026-02-04

💼 32 ⚡ 82

FastSLM addresses the token explosion problem in speech-language models through a Hierarchical Frame Q-Former that progressively compresses acoustic features across multiple temporal scales, reducing speech representation to just 1.67 tokens/second—a 93% reduction compared to standard approaches. This extreme token efficiency enables MLLMs to handle long-form speech processing that would otherwise be computationally prohibitive, making end-to-end speech understanding practical for resource-constrained deployment scenarios.

Video-OPD: Efficient Post-Training of Multimodal Large Language Models for Temporal Video Grounding via On-Policy Distillation

cs.CV updates on arXiv.org · 2026-02-04

💼 22 ⚡ 78

Video-OPD introduces an on-policy distillation framework for temporal video grounding that addresses the computational inefficiency and sparse reward limitations of existing GRPO-based methods by leveraging a frontier teacher to provide dense, token-level supervision via reverse KL divergence. The approach maintains on-policy optimization benefits while converting sparse episode-level feedback into fine-grained step-wise learning signals, with an additional Teacher-Validated Disagreement Focusing (TVDF) mechanism for optimized curriculum training. This represents a practical efficiency gain for post-training multimodal LLMs on video understanding tasks while preserving the distributional alignment critical for RL-based approaches.

This approach is particularly impactful for vision-language models where visual sequences contain significantly more low-information-density tokens than text, making compression-aware drafting substantially more effective than existing methods.

video-SALMONN S: Memory-Enhanced Streaming Audio-Visual LLM

cs.CV updates on arXiv.org · 2026-02-04

💼 22 ⚡ 88

**video-SALMONN S** introduces test-time training (TTT) as a novel streaming memory mechanism for long-duration audio-visual understanding, enabling efficient processing of 3+ hour videos at 1 FPS by continuously consolidating short-term multimodal representations into long-term parameter-embedded memory. Key innovations include a TTT_MEM layer with long-span prediction objectives, modality-aware memory readers, and a two-stage training scheme that outperforms non-streaming baselines under comparable memory budgets. This approach addresses a critical bottleneck in video agents by replacing traditional KV-cache limitations with learnable parameter-based memory, opening new directions for efficient long-context multimodal models.

PolarMem: A Training-Free Polarized Latent Graph Memory for Verifiable Multimodal Agents

cs.AI updates on arXiv.org · 2026-02-04

💼 15 ⚡ 72

PolarMem addresses a critical gap in multimodal agent memory by introducing a training-free system that converts probabilistic vision-language model outputs into verifiable logical constraints through non-parametric distributional partitioning, explicitly encoding negative facts via polarized graph topology with inhibitory connections. This approach resolves the epistemic asymmetry problem where current dense memories conflate semantic similarity with factual truth, enabling logic-dominant retrieval that supports verifiable long-horizon reasoning. The training-free nature and discrete constraint representation make this particularly valuable for safety-critical applications where agents must distinguish between "not found" and "found to be false"—a distinction essential for reliable autonomous decision-making.

MACD: Model-Aware Contrastive Decoding via Counterfactual Data

cs.AI updates on arXiv.org · 2026-02-04

💼 22 ⚡ 78

MACD addresses hallucination in Video-LLMs by replacing random perturbations in contrastive decoding with model-guided counterfactual data generation—leveraging the model's own feedback to identify and surgically modify object regions most responsible for hallucinations rather than arbitrary frame-level changes. This targeted approach enables more precise alignment between counterfactual construction and model weaknesses, forcing the decoder to select tokens grounded in visual evidence. The method represents a meaningful shift from black-box perturbation strategies to interpretable, model-aware inference that could improve reliability in vision-language reasoning tasks.
