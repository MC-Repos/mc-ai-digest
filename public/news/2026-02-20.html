EntropyPrune: Matrix Entropy Guided Visual Token Pruning for Multimodal Large Language Models

cs.CV updates on arXiv.org · 2026-02-20

💼 23 ⚡ 78

EntropyPrune introduces a principled, model-agnostic approach to visual token pruning in MLLMs by leveraging matrix entropy to identify an "Entropy Collapse Layer" where information content sharply drops, enabling optimal pruning stage selection without heuristic layer selection. The framework quantifies individual token information value independently of attention maps, potentially improving both interpretability and transferability across different multimodal architectures while reducing substantial inference costs from processing hundreds of visual tokens per image.

Xray-Visual demonstrates effective scaling of unified vision models by combining 15B image-text and 10B video-hashtag pairs from social media with a three-stage training pipeline (MAE pre-training, semi-supervised hashtag classification, and CLIP-style contrastive learning) on Vision Transformer backbones with efficient token reorganization. This work highlights how industrial-scale multimodal datasets with proper denoising strategies can improve both image and video understanding without requiring perfectly labeled data.

DDiT: Dynamic Patch Scheduling for Efficient Diffusion Transformers

cs.AI updates on arXiv.org · 2026-02-20

💼 32 ⚡ 72

DDiT introduces dynamic patch scheduling for Diffusion Transformers, adaptively varying patch sizes across denoising timesteps—coarser patches early for global structure modeling, finer patches later for local detail refinement—achieving significant computational savings without compromising generation quality. This test-time optimization exploits the insight that content complexity requirements differ across diffusion steps, offering a practical efficiency gain for image and video generation without architecture modifications. The approach addresses a fundamental inefficiency in current DiT implementations where fixed tokenization wastes compute on unnecessary fine-grained tokens during coarse-detail-dominant early diffusion phases.

VP-VAE: Rethinking Vector Quantization via Adaptive Vector Perturbation

cs.AI updates on arXiv.org · 2026-02-20

💼 23 ⚡ 72

VP-VAE reframes vector quantization as adaptive latent perturbation rather than discrete codebook indexing, using Metropolis-Hastings sampling to inject distribution-consistent noise during training while decoupling representation learning from discretization. This approach eliminates codebook collapse and training instability endemic to standard VQ-VAE while maintaining robustness to quantization error at inference time. The key innovation is treating quantization as a learned stochastic process rather than a hard operation, potentially enabling more stable training of discrete generative models at scale.

Resp-Agent: An Agent-Based System for Multimodal Respiratory Sound Generation and Disease Diagnosis

cs.AI updates on arXiv.org · 2026-02-20

💼 23 ⚡ 82

Resp-Agent introduces an agent-based framework that addresses two critical bottlenecks in respiratory sound diagnosis: information loss from spectrogram conversion and severe data scarcity through an Active Adversarial Curriculum Agent that actively identifies diagnostic weaknesses and schedules targeted synthesis. The system innovates with a Modality-Weaving Diagnoser that preserves millisecond-level acoustic transients by fusing EHR data with audio tokens via Strategic Global Attention, and employs a Flow Matching Generator for synthetic data augmentation. This closed-loop, curriculum-driven approach represents a significant shift from static diagnostic pipelines toward adaptive multimodal systems that can handle severe class imbalance in clinical audio datasets.
