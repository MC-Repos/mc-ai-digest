cs.AI updates on arXiv.org · 2026-02-19

💼 28 ⚡ 35

This survey introduces NLP-PRISM, a comprehensive framework evaluating privacy vulnerabilities across six dimensions (data collection, preprocessing, visibility, fairness, computational risk, and regulatory compliance) in social media NLP applications that frequently process PII and behavioral data. The analysis of 203 papers reveals a critical performance-privacy tradeoff: transformer models drop 1-23% in F1-score when privacy-preserving fine-tuning is applied, establishing baseline degradation metrics for practitioners. The framework systematically maps privacy risks across six common NLP tasks, providing researchers and developers with actionable vulnerability assessment methodology for building privacy-aware social media analytics systems.

cs.AI updates on arXiv.org · 2026-02-19

💼 23 ⚡ 72

Fly0 decouples semantic understanding from control by using MLLMs for high-level instruction grounding to 2D coordinates, then offloading geometric planning to a dedicated depth-based 3D localization and trajectory generation pipeline—eliminating the latency and instability of end-to-end MLLM control. This modular architecture enables zero-shot aerial navigation with robustness to visual occlusion and reduced computational overhead through single-pass semantic inference rather than continuous model deployment.

FUTURE-VLA: Forecasting Unified Trajectories Under Real-time Execution

cs.AI updates on arXiv.org · 2026-02-19

💼 28 ⚡ 72

FUTURE-VLA addresses the real-time deployment challenge for vision-language models on robots by unifying long-horizon control and future forecasting into a single sequence-generation task, using temporally adaptive compression to maintain constant inference latency despite processing extensive multi-view histories. The key innovation is latent-space autoregression that simultaneously predicts actionable robot dynamics and visual future states in one forward pass, enabling prediction-guided human-in-the-loop control without the typical latency bottleneck of processing long video contexts. This approach is particularly significant for embodied AI systems requiring real-time decision-making, as it decouples inference cost from history length through intelligent spatiotemporal compression rather than naive token reduction.

Resp-Agent introduces an agent-based framework addressing critical limitations in respiratory disease diagnosis by combining an Active Adversarial Curriculum Agent that dynamically identifies diagnostic weaknesses and schedules targeted data synthesis, with a Modality-Weaving Diagnoser that fuses EHR data and raw audio tokens via Strategic Global Attention to preserve transient acoustic events lost in spectrogram conversion. The system tackles both information loss (through multimodal fusion preserving millisecond-level details) and severe class imbalance (via adaptive Flow Matching generation), moving beyond static pipelines to a closed-loop learning approach that could significantly improve robustness in clinical respiratory auscultation tasks.

cs.AI updates on arXiv.org · 2026-02-19

💼 25 ⚡ 72

MedProbCLIP extends vision-language foundation models (like CLIP) to medical imaging by replacing deterministic embeddings with probabilistic Gaussian representations, enabling uncertainty quantification and many-to-many correspondences between chest X-rays and radiology reports. The framework uses variational information bottlenecks to prevent overconfident predictions and incorporates multi-view image/multi-section text encoding for clinically-aligned fine-grained supervision, directly addressing reliability concerns critical for high-stakes biomedical applications. This probabilistic approach enables more robust retrieval systems that can express confidence in radiograph-report matches—essential for clinical deployment where false positives carry significant diagnostic risk.

This approach is particularly significant for production video generation systems where batch processing similar prompts is common.

cs.CV updates on arXiv.org · 2026-02-19

💼 22 ⚡ 62

UncL-STARK introduces dynamic depth adaptation for transformer-based visual trackers, allowing variable encoder-decoder truncation at inference time based on frame uncertainty rather than executing full-depth processing on every frame. The approach uses random-depth training with knowledge distillation to ensure robustness at intermediate depths, while deriving lightweight uncertainty estimates directly from corner localization heatmaps to guide a feedback-driven depth selection policy. This architecture-preserving method delivers computational efficiency gains on temporally coherent video sequences without modifying the underlying tracker network or adding auxiliary components.
